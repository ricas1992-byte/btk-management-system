# מעבר לקלידים – Beyond the Keys
## מערכת ניהול פנימית / BTK Management System

מערכת ניהול מקיפה לכתיבה, למידה, וניהול מסמכים ופרויקטים.

---

## 🎨 תכונות עיקריות

### ✅ מצב בהיר ומצב כהה
- מצב בהיר כברירת מחדל - עיצוב נקי ומקצועי
- מצב כהה זמין בלחיצת כפתור (💡/🌙)
- העדפת המשתמש נשמרת אוטומטית

### 📦 5 מודולים מלאים

1. **📊 לוח מצב (Dashboard)** - סקירה כללית של כל הפעילויות
2. **📄 ספריית מסמכים** - עורך מסמכים מתקדם עם יצוא HTML
3. **📚 סביבת למידה** - העלאת חומרי למידה, הדגשות ותקצירים
4. **✍️ סטודיו כתיבה** - כתיבה מובנית (ספר → חלק → פרק → מקטע)
5. **⚙️ שכבת ניהול** - ניהול משימות, סקירות ומדדים

---

## 🚀 התחלת עבודה

### דרישות מקדימות
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- שרת HTTP מקומי (למשל: Python, Node.js, או Live Server)

### הפעלה

#### אופציה 1: Python
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### אופציה 2: Node.js
```bash
npx http-server
```

#### אופציה 3: VS Code Live Server
1. התקן את התוסף "Live Server"
2. לחץ ימני על `index.html` → "Open with Live Server"

## 💾 ניהול נתונים

המערכת משתמשת בגישת "Git-First":
- נתונים נשמרים ב-`localStorage` (לדמו בדפדפן)
- קבצי JSON מועתקים ל-`data/` (לניהול Git)
- בעתיד: אינטגרציה עם Git להיסטוריה מלאה
